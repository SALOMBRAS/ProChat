# Aplicação web do ChatPro

Workspace Next.js do frontend acadêmico do ChatPro.

Nesta fase, a aplicação contém somente uma página neutra que confirma a fundação técnica. Não há autenticação, Supabase, integração com WhatsApp, CRM, PWA ou deploy configurados.

Execute os comandos a partir da raiz do repositório:

```powershell
npm run dev:web
npm run lint
npm run typecheck
npm run build:web
```
